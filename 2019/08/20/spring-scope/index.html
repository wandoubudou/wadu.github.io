Spring Scope

@Scope是用来描述Spring Bean的创建方法，主要有如下几种方法

  1. @Scope(“singleton”) bean中只有一个实例 默认方法
  2. @Scope(“prototype”) bean中多个实例，每次调用都会新建一个bean
  3. @Scope(“request”) Web项目中 给http request创建一个bean实例
  4. @Scope(“session”) Web项目中 给http session创建一个bean实例
